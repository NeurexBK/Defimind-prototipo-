# DefiMind — Protótipo Interativo

Demo clicável da interface do DefiMind: conta digital que rende em dólar (USDC).

## Arquivos

| Arquivo | O que é |
|---|---|
| `index.html` | **Página principal** — o cliente escolhe entre a versão computador e a versão celular (as duas demos completas embutidas) |
| `versao-desktop.html` | Só a versão computador (navegação no topo: Início, Cartão, Investir, Portfólio, Conquistas, Assistente) |
| `versao-celular.html` | Só a versão celular (app com abas embaixo, moldura de iPhone) |

## Como usar

Abra `index.html` em qualquer navegador — não precisa de servidor, tudo funciona offline.

Para publicar na internet (link público para investidores): ative o **GitHub Pages** em
*Settings → Pages → Deploy from a branch → main*. O site ficará disponível em
`https://neurexbk.github.io/Defimind-prototipo-/`.

## O que a demo tem

- Tudo em **USDC** (stablecoin, 1 USDC = 1 dólar) — sem BTC/ETH
- **Cartão** com bloquear/desbloquear, ver dados, cartão virtual, limites e cashback de 1% em USDC
- **Caixinha do dia a dia** rendendo ao vivo (contador em tempo real, 8% ao ano)
- **3 potes de investimento** com linguagem simples e simulador de rendimento
- **Conquistas**: sequência 🔥, XP, níveis e medalhas
- **Assistente IA** em formato de chat
- Identidade visual de banco (verde-cofre + dourado), sem roxo/azul — paleta dos gráficos validada para daltonismo e contraste

> Dados ilustrativos, apenas para demonstração.
